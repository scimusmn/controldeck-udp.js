controldeck-udp.js

control HTML5 Presentations
with node.js and UDP messages

The original work on controldeck.js was built and designed by John Polacek
ControlDeck-udp.js is a fork of ControlDeck.js.
It has been modified for UDP messaging to facilitate kiosk and show control use by Bryan Kennedy
for use at the Science Museum of Minnesota

Source code and documentation at
github.com/scimusmn/controldeck-udp.js

Below are various HTML5 slide presentation demos. Open the controller in a new window and use the buttons to update the slides opened in a different window.
